# Paradise Nursery Shopping Application

This is a dynamic web application for an online plant shop called Paradise Nursery. It allows users to browse a variety of houseplants, view details such as images, names, descriptions, and prices, add plants to a shopping cart, and manage cart items (add, remove, adjust quantities). The application emphasizes interactive features like a navigation bar, dynamically updating cart quantities, and calculating total costs.

## Features
- Landing page with company name and "Get Started" button.
- Product listing page with plants grouped into categories (Indoor, Outdoor, Succulents), each with thumbnail, name, price, and "Add to Cart" functionality.
- Shopping cart page with item management (increase/decrease quantity, delete items), totals, and checkout placeholder.
- Redux-based state management for cart.
- Responsive navigation bar with cart icon showing item count.

## Technologies Used
- React
- Redux Toolkit
- React Router
- CSS

## Setup
1. Clone the repository.
2. Run `npm install`.
3. Add plant images to `public/images/` (e.g., plant1.jpg, plant2.jpg, etc.).
4. Run `npm start` to launch the app.
